Ejemplo.
Ejemplo.


Cuando se convierten tipos de datos con diferentes lugares decimales, algunas veces el valor del resultado es truncado y otras veces es redondeado. La siguiente tabla muestra el comportamiento.
Ejemplo.


Por ejemplo, el resultado de la siguiente conversión es 10.
SELECT CAST(10.6496 AS int);
Cuando se convierten tipos de datos en los cuales el tipo de dato objetivo tiene menos lugares decimales que la fuente, el valor es redondeaso. For ejemplo, el resultado de la siguiente conversión es $10.3497:
SELECT CAST(10.3496847 AS money);
