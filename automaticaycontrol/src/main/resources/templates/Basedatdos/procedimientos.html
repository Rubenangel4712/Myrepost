Procedimientos almacenados

Un procedimiento almacenado es un conjunto de código puro del lenguaje que se utiliza dentro de las bases de datos como por ejemplo: create table, insert into, Select from, y que se almacena físicamente en la base de datos y de la misma forma se exportan cuando creamos una copia de seguridad de nuestra base de datos. Los procedimientos almacenados se basan en una estructura definida por SQL SERVER:
Iniciamos con:
• CREATE PROCEDURE nombre_del_procedimiento @variables tipo(longitud)
• AS
• BEGIN
• Todo el código puro de base de datos
• END
 Ejemplo. Crear un procedimiento para insertar un nuevo producto en la tabla PRODUCT. (SaleCo)

El procedimiento creado puede ser verificado en SaleCo >> Programación.
Pare ejecutar el procedimiento hacemos: INSERT_PRODUCT '35RR721', 'SERRUCHO', '2014/03/06', 25, 15, 258, 25, 25600
