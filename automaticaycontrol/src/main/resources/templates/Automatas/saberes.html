Esta sección tiene como objetivo hacer un repaso sobre el manejo de algunos conceptos matemáticos que serán de utilidad para el desarrollo del curso. NO es una actividad evaluativa por lo cual no se tendrá que entregar ninguna evidencia de esta.

Ejercicios sobre inducción matemática.

  1. Demostrar que la suma de los n primeros números naturales es igual a n(n+1)/2.
  2. Probar que , ∀n ∈ Ν : 1⋅3+2⋅4+3⋅5+ …+n(n+2) = n(n+1).(2n+7)/6.
  3. Determinar si el producto de 3 números impares consecutivos es siempre divisible por 6.

Ejercicios sobre teoría de conjuntos.

  1. Determinar, en el conjunto de los números enteros, cuáles de los siguientes conjuntos son iguales.
    A ={x : x es par y x2 es impar}
    B = {x : Ǝ y, y ∈ Z ∧ x = 2y}
    C = {1, 2, 3}
    D = {0, 2,-2, 3,-3, 4,-4, . . .}
    E = {2x : x ∈ Z}
    F = {3, 3, 2, 1, 2}
    G = x : x3 - 6x2 - 7x - 6 = 0

  2. ¿Es B = {1, 2, 3} un subconjunto de A = {x ∈ R : x2 - 3x + 2 = 0}?

  3. Para cada uno de los siguientes conjuntos , elegir un conjunto universal y un predicado apropiado para definirlo.
    1. El conjunto de los enteros entre 0 y 100.
    2. El conjunto de los enteros positivos impares.
    3. El conjunto de los múltiplos de 10.

Ejercicios sobre lógica proposicional.

  1. Cuál de las siguientes fórmulas representa la proposición “Llegará en el tren de las 8:15 o en el de las 9:15, si llega en el primero, entonces tendrá tiempo para visitarnos”. Donde:
    p expresa “Llegará en el tren de las 8:15”
    q expresa “Llegará en el tren de las 9:15”
    r expresa “Tendrá tiempo para visitarnos”

    1. ¬p → q v r
    2. p v q → r
    3. (p → q) ∧ (p ∧ r)
    4. p v ¬q → r
    5. (p v q) ∧ (p → r)

  2. Sean las fórmulas F = (p → q) v ((¬p v q)∧(q → r)) y G = (p → q). Determinar si las dos fórmulas son equivalentes.

  3. Repita el ejercicio anterior haciendo uso de tablas de verdad.
