Lenguajes y Expresiones Regulares

Una expresión regular (e.r) es una notación normalizada para representar lenguajes regulares o lenguajes de gramática tipo 3. Las e.r permiten escribir con exactitud y sencillez cualquier lenguaje regular. Además de todos los símbolos del alfabeto ∑ también se utilizan los símbolos Ф, λ, +, ., * y ().
  • Ф: el lenguaje vacío.
  • λ: la palabra vacía.
  • +: representa la unión.
  • .: concatenación (aunque suele no escribirse).
  • *: representa el cierre (estrella de Kleene).
  • (): modifican las prioridades de los demás operadores.
Una e.r se puede definir de acuerdo con los siguientes criterios:
  1. Ф es una e.r que representa al lenguaje vacío (no tiene palabras) LФ = Ф
  2. λ es una e.r que representa al lenguaje Lλ = {λ}.
  3. a ϵ ∑ es una e.r que representa al lenguaje La = {a}.
  4. Si α y β son e.r entonces α + β también lo es y representa al lenguaje L(α+β) = Lα U Lβ
  5. Si α y β son e.r entonces αβ también lo es y representa al lenguaje Lαβ = LαLβ
  6. Si α es una e.r entonces α* también lo es y representa al lenguaje Lα* = U∞(i=0) Lαi que también se puede representar como α*= ∑∞(i=0)αi

Solo son e.r aquellas que puedan ser definidas utilizando estos seis puntos. La prioridad de las operaciones, que puede modificarse utilizando paréntesis es de mayor a menor: * . +

Ejemplo.

Sea ∑ = {0, 1}
  1. 01 + 001 es una e.r que representa al lenguaje L = {01,001}
  2. 0*10* es una e.r que representa a cualquier cadena binaria en la que hay un solo 1, L = {0n10m /n,m≥0 }
Sea ∑ = {a, b, c}
  1. a(b + c)* representa a cualquier cadena que empiece por a.
  2. (b + c)* representa al lenguaje universal definido sobre el alfabeto.
Álgebra de las expresiones regulares.

Propiedades de la unión.
1. Asociativa (α + β) + ϒ = α + (β + ϒ)
2. Conmutativa α + β = β + α
3. Elemento neutro (Ф) α + Ф = α
4. Idempotencia α + α = α

Propiedades de la concatenación.
1. Asociativa (αβ)ϒ = α(βϒ)
2. Distributiva respecto a la unión. α(β + ϒ) = αβ + αϒ
3. Elemento neutro (λ) αλ = λα = α
4. αФ = Ф
5. No es conmutativa

Propiedades del cierre de Kleene(*).
  1. λ* = λ
  2. Ф* = λ
  3. α* α* = α*
  4. α*α = αα*
  5. (α*)* = α*
  6. α* = λ + αα*
  7. α* = λ + α + α2 + … + αn + αn+1 α*
  8. A+ = A*.A = A.A* (A+ se denomina clausura positiva)
  9. A* A* = A*
  10. (A* )n = A* para todo n ≥1
  11. (A*)* = A*
  12. A+ A+⊆ A+
  13. (A+)* = A*
  14. (A+)+ = A+
  15. Si A y B son lenguajes sobre ∑*, entonces (A U B)* = (A*B*)*

Ejemplos.

Sea ∑ = {a, b}. Los siguientes son lenguajes regulares sobre ∑.
  1. El lenguaje A de todas las cadenas que tienen exactamente una a:

    A= {b}*. { a } . { b }*

    Una expresión regular para este lenguaje es: b*ab*
  2. El lenguaje B de todas las cadenas que comienzan con b:

    B= b(a U b )*

    Una expresión regular para este lenguaje es: b*ab*
  3. El lenguaje C de todas las cadenas que contienen la cadena ba:

    C= {a, b}*. { ba } . {a, b }*

    Una expresión regular para este lenguaje es: C= (a, b)*. ba . (a, b )*
  4. Dado el alfabeto ∑ = {a, b, c}, una expresión regular que representa al lenguaje

    A= ({a} U {b}* . {a}*. {bc}*

    Una expresión regular para este lenguaje es: (aUb*)a*(bc)*
  5. Dado el alfabeto ∑ = {a, b}

    (λ U a)*(a U b)*(ba)*

    es una expresión regular que representa al lenguaje

    A= ({λ} U {a})*.({a} U {b})*.{ba}*

La representación de lenguajes regulares por medio de e.r no es única. Es posible que haya varias expresiones regulares diferentes para el mismo lenguaje. Por ejemplo, b(a U b)* y b(b U a)* representan el mismo lenguaje.

Ejemplos.

Encontrar expresiones regulares que representen los siguientes lenguajes definidos sobre el alfabeto ∑ = {a, b}.
  1. Lenguaje de todas las cadenas que comienzan con el símbolo b y terminan con el símbolo a.

    b(a U b)*a

  2. Lenguaje de todas las cadenas que tienen un número par de símbolos (cadenas de longitud par).

    (aa U ab U ba U bb)*. Otra e.r para este lenguaje es:((a U b)(a U b))*

    .
  3. El lenguaje de todas las cadenas que tienen un número par de aes.
    1. b*(b*ab*ab*)*
    2. (ab*a U b)*
    3. (b*ab*ab*)* U b*

Ejemplos.

Encontrar expresiones regulares que representen los siguientes lenguajes definidos sobre el alfabeto ∑ = {0, 1}.
  1. Lenguaje de todas las cadenas que tienen exactamente dos ceros.

    1*01*01*

  2. Lenguaje de todas las cadenas cuyo penúltimo símbolo de izquierda a derecha es un 0.

    (0 U 1)*0(0 U 1).

Ejemplo.

Sea ∑ = {0, 1}, encontrar una e.r que represente al lenguaje de todas las cadenas que no contienen ceros consecutivos.
La condición de que no hayan ceros consecutivos implica que todo cero debe estar seguido necesariamente de un uno, excepto un cero al final de la cadena. Por lo tanto, las cadenas de este lenguaje se obtienen concatenando unos con bloques 01 de todas las formas posibles. Hay que tener en cuenta, además, que la cadena puede terminar ya sea en 1 o en 0.

(1 U 01)* U (1 U 01)* 0

Ejemplo.

Sea ∑ = {a, b, c}, encontrar una e.r que represente el lenguaje de todas las cadenas que no contienen la cadena bc. Una b puede estar seguida solamente de otra b o de una a, mientras que las aes y las ces pueden estar seguidas de cualquier símbolo. Teniendo en cuenta todas las restricciones y posibilidades, se llega a la siguiente e.r.

(a U c U b+a)*b*

La condición de que no aparezca la cadena bc significa que una c puede estar precedida solamente de una a y de otra c. Siguiendo esta descripción, se llega a:

c*(b U ac*)*
