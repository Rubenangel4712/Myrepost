Simplificación de Gramáticas Independientes del Contexto

En una GIC se pueden tener árboles incontrolablemente tupidos o inútilmente profundos. Para evitar situaciones como estas, la idea es encontrar algún modelo o forma estándar para las producciones.
Como primer paso en el desarrollo de este modelo, es necesario limpiar las gramáticas para eliminar las producciones y símbolos inútiles.

Ejemplo.

Supóngase la siguiente GIC:
S → Aa | B | D
B → b
A → aA | bA| B
C → abd
Obsérvese que C nunca formará parte de una derivación que parta del símbolo inicial, es decir, no existe S⇒* αCβ para toda cadena α y β de (N U ∑)*. Por consiguiente, el símbolo C y la producción C → abd son inútiles en el sentido que nunca podrán contribuir a la generación de una cadena de L(G). Por otro lado, el símbolo D se obtiene a partir de S pero nunca genera una cadena de símbolos terminales y, por tanto, nunca formará parte de una derivación de una cadena de terminales. Así que, D también es inútil, aunque por distinta razón. Además, el símbolo B es, en cierto modo redundante, ya que deriva únicamente un símbolo terminal pudiéndose sustituir por las producciones S → b y A →b y de esta forma eliminar a B. Finalmente, eliminando la producción C → abd, el símbolo d no aparecerá en ninguna de las cadenas generadas por las producciones.
De esta forma, el método deberá eliminar primero los no terminales que no deriven cadenas de terminales, tales como D en este ejemplo.

Definiciones.
  1. Una variable A es terminable si existe w ∈ ∑* tal que A ⇒* w. En particular, si A → λ es una producción entonces A es terminable.
  2. Una variable A es alcanzable o accesible si existen u, v ∈ (V U ∑ )* tales que S ⇒* uAv. La variable inicial S es alcanzable por definición.
  3. A es una variable inútil si no es alcanzable o no es terminable.
Existen algoritmos para detectar todas las variables inútiles de una GIC que permiten construir una gramática G’ equivalente a una gramática G dada, de tal manera que G’ no tenga variables inútiles.

Algoritmo para encontrar las variables terminables.

    INICIALIZAR:
        TERM:= {A ∈ V∶ ∃ producción A → w, w ∈ ∑*}
    REPETIR:
        TERM:= TERM U {A ∈ V∶ ∃ producción A → w, w ∈ (∑ U  TERM)*}
    HASTA:
        No se añaden nuevas variables a TERM

Ejemplo.

Encontrar las variables terminables de la siguiente gramática.
1. S → ACD | bBd |ab
2. A → aB | aA | C
3. B → aDS | aB
4. C → aCS | CB | CC
5. D → bD | ba
6. E → AB | aDb
Solución.
        TERM1 = {S, D}: D se agrega a partir de la regla 5 ya que genera una cadena sin no terminales (ba).
        TERM2 = {S, D} U {B, E} = { S, D, B, E}: como ya está D, a partir de 6 (aDb) se agrega a E y a partir de 3 (aDS) se agrega a B.
        TERM3 = { S, D, B, E} U {A} = { S, D, B, E, A}: como ya está B, a partir de 2 (aB) se agrega a A.
        TERM4 = { S, D, B, E, A} U { } = { S, D, B, E, A}: ya no hay más terminales para agregar.
        TERM =  { S, D, B, E, A}: éstos serían los nuevos símbolos gramaticales.
Símbolos no terminables: {C}
Con esta nueva gramática, las reglas de producción quedarían como:
S → ACD | bBd |ab
A → aB | aA
B → aDS | aB
D → bD | ba
E → AB | aDb

Ejemplo.

Gramática original.
S → ABaC | SB | λ
A → aAB | bB
B → Ea | bB |λ
E → Ee

Gramática simplificada (paso 1).
S → SB | λ
A → aAB | bB
B → bB | λ

Algoritmo para encontrar las variables alcanzables o accesibles. 

        INICIALIZAR:
	        ALC:= {S}
        REPETIR:
	        ALC:= ALC U {A ∈ V∶ ∃ producción B → uAv, B ∈ ALC y u, v∈(V U ∑)*}
        HASTA:
	        No se añaden nuevas variables a ALC

Ejemplo.

Encontrar las variables alcanzables de la siguiente gramática.
S → aS | AaB |ACS
A → aS | AaB |AC
B → bB | DB |BB
C → aDa | ABD |ab
D → aD | DD |ab
E → FF | aa
F → aE | EF 

    ALC1 = {S}
    ALC2 = {S} U {A, B, C} = {S, A, B, C}
    ALC3 = {S, A, B, C} U {D} = {S, A, B, C, D}
    ALC4 = {S, A, B, C, D} U { } = {S, A, B, C, D}
    ALC = {S, A, B, C, D}
Variables no alcanzables: {E, F}
Eliminando las variables no alcanzables, las reglas de producción quedarán:
S → aS | AaB |ACS
A → aS | AaB |AC
B → bB | DB |BB
C → aDa | ABD |ab
D → aD | DD |ab

Ejemplo.

Eliminar los símbolos inalcanzables de la siguiente gramática.
S → SB | λ
A → aAB | bB
B → bB | λ

Paso 1. Como S es símbolo de inicio, y su producción es S → SB | λ, entonces B será alcanzable: ALC = {S, B}.
Paso 2. Como B es alcanzable, entonces todo lo que tenga al lado derecho a B o S también lo será. Con la producción B → bB | λ se tendrá entonces que:
ALC = {S, B, b}.
Paso 3. Como ya no hay más símbolos alcanzables, los que quedan serán símbolos inalcanzables y deberán ser eliminados de la gramática. De esta forma, las reglas quedarán como:
S → SB | λ
B → bB | λ

Eliminación de las producciones λ.


Definiciones.
  1. Una producción de la forma A →λ se llama producción λ.
  2. Una variable A se llama anulable si A ⇒* λ.
Algoritmo para encontrar las variables anulables.
        INICIALIZAR:
	        ANUL:= { A ∈V:A  → λ es una producción }
        REPETIR:
            ANUL:= ANUL U { A ∈V:∃ producción A  → w ,w ∈(ANUL )^ }
        HASTA:
	        No se añaden nuevas variables a ANUL

Ejemplo.

Eliminar las producciones λ de la siguiente gramática:
S → AB | ACA | ab
A → aAa | B | CD
B → bB | bA
C → cC |λ
D → aDc | CC |ABb

Solución. Primero se encuentran las variables anulables con λ.

ANUL1 = {C}
ANUL2 = {C} U {D}={C, D}
ANUL3 = {C, D} U {A} = {C, D, A}
ANUL4 = {C, D, A} U {S}={C, D, A, S}
ANUL5 = {C, D, A, S} U { } = {C, D, A, S}
ANUL = {C, D, A, S}

Se eliminan de G las producciones λ.

Original (G) _Elimina anul_ Equivalente (G')
S → AB | ACA | ab
A → aAa | B | CD
B → bB | bA
C → cC |λ
D → aDc | CC | ABb
S → AB | ACA |ab | B | CA | AC |AA | A |C
A → aAa | B |CD | aa | C | D
B → bB | bA | b
C → cC |c
D → aDc | CC |ABb | ac |C | Bb

De la original se obtiene la equivalente reemplazando por λ todas las posibles combinaciones de las variables anulables (negrita) al lado izquierdo.

Eliminación de las producciones unitarias.


Definiciones.
  1. Una producción de la forma A →B donde A y B son variables, se llama una producción unitaria.
  2. El conjunto unitario de una variable A (también llamado conjunto cadena de A) se define de la siguiente manera:
    UNIT(A):= { X ∈ V: ∃ una derivación A ⇒* X que usa únicamente producciones unitarias }.
Por definición, A ∈ UNIT(A)
Algoritmo para encontrar las producciones unitarias.
        INICIALIZAR:
	        UNIT(A)  :=  { A }
        REPETIR:
            UNIT(A)  :=  UNIT(A) U { X ∈ V: ∃ una producción Y  → X con Y ∈ UNIT(A)}
        HASTA:
	        No se añaden nuevas variables a UNIT(A)

Ejemplo.

Eliminar las producciones unitarias de las siguientes gramáticas:

  1. S → AS | AA | BA | λ
    A → aA | a
    B → bB | bC | C
    C → aA | bA | B | ab

    Solución. Primero se encuentran los conjuntos de variables unitarias:
    UNIT1(S) = {S}
    UNIT2(S) = {S} U { }={S}
    UNIT(S) = {S}

    UNIT1(A) = {A}
    UNIT2(A) = {A} U { }={A}
    UNIT(A) = {A}

    UNIT1(B) = {B}
    UNIT2(B) = {B} U {C} = {B, C}
    UNIT3(B) = {B, C} U { }={B, C}
    UNIT(B) = {B, C}

    UNIT1(C) = {C}
    UNIT2(C) = {C} U {B}={C, B}
    UNIT3(C) = {C,B} U { }={C, B}
    UNIT(C) = {C, B}

    Eliminando las producciones unitarias se obtiene:

    Original (G) _Elimina unit_ Equivalente (G')
    S → AS | AA | BA | λ
    A → aA | a
    B → bB | bC | C
    C → aA | bA | B | ab
    S → AS | AA | BA | λ
    A → aA | a
    B → bB | bC | aA | bA | ab
    C → aA | bA | B | ab | bB | bC


  2. S → ACA | CA |AA | A |C | λ
    A → aAa | aa | B | C
    B → cC | D | C
    C → bC
    D → aA | λ

Solución. Primero se encuentran los conjuntos de variables unitarias:

UNIT1(S) = {S}
UNIT2(S) = {S} U {A} = {S, A}
UNIT3(S) = {S, A} U {C} = {S, A, C}
UNIT4(S) = {S, A, C} U {B } = {S, A, C, B}
UNIT5(S) = {S, A, C, B} U {D} = {S, A, C, B, D}
UNIT6(S) = {S, A, C, B, D} U { } = {S, A, C, B, D}
UNIT(S)={S, A, C, B, D}

UNIT1(A) = {A}
UNIT2(A) = {A} U {B} = {A, B}
UNIT3(A) = {A, B} U {C} = {A, B, C}
UNIT4(A) = {A, B, C} U {D} = {A, B, C, D}
UNIT5(A) = {A, B, C, D} U { } = {A, B, C, D}
UNIT(A) = {A, B, C, D}

UNIT1(B) = {B}
UNIT2(B) = {B} U {C} = {B, C}
UNIT3(B) = {B, C} U {D} = {B, C, D}
UNIT4(B) = {B, C, D} U { } = {B, C, D}
UNIT(B)={B, C, D}

UNIT1(C) = {C}
UNIT2(C) = {C} U { } = {C}
UNIT(C)={C}

UNIT1(D) = {D}
UNIT2(D) = {D} U { } = {D}
UNIT(D) = {D}

Eliminando las producciones unitarias se obtiene:

Original (G) _Elimina unit_ Equivalente (G')
S → ACA | CA |AA | A | C | λ
A → aAa | aa | B | C
B → cC | D | C
C → bC
D → aA | λ

A → aAa | aa | cC | aA | bC | λ
B → cC | aA | bC | λ
C → bC
D → aA | λ
