Forma Normal de Chomsky (FNC)

Una GIC G está en FNC si satisface:
  1. G no tiene variables inútiles.
  2. G no tiene producciones λ (excepto posiblemente S → λ ) .
  3. Todas las producciones son de la forma: A → a (producciones simples) o de la forma A → BC (producciones binarias). ABC ∈ N y a ∈ ∑
En particular, una gramática FNC no tiene producciones unitarias.

Procedimiento para obtener la FNC de una gramática dada.
  1. Eliminar las variables no terminables.
  2. Eliminar las variables no alcanzables.
  3. Eliminar las producciones λ (excepto S → λ, si existe).
  4. Eliminar las producciones unitarias.
  5. Las producciones de la forma A → w con w ≥ 2 se substituyen con una nueva producción T → w y las producciones con más de dos variables no terminales se substituyen con nuevas producciones binarias.

Ejemplo.
S → aB | bA
A → a | aS | bAA
B → b | bS | aBB

Obsérvese que esta gramática ya se encuentra libre de anomalías. Esta gramática genera el lenguaje de las cadenas que tienen el mismo número de aes que de bes.

En esta gramática, las únicas producciones que se encuentran en FNC son las indicadas aquí con negrita:
S → aB | bA
A → a | aS | bAA
B → b | bS | aBB

Las otras producciones tienen mezclados símbolos terminales con símbolos no terminales. Para evitar esto se introducen nuevas producciones así:
S → CB | DA
A → a | CS | DAA
B → b | DS | CBB
C → a
D → b

Las producciones con más de dos símbolos no terminales se substituyen así:
S → CB | DA
A → a | CS | DE
B → b | DS | CF
C → a
D → b
E → AA
F → BB

Esta gramática ya se encuentra en la FNC.
Ejemplo. Encontrar la FNC de la siguiente gramática:
S → aBA | bA | a
A → ab | AbA
B → b

Solución:
Se encuentran los terminales que ya están en esta forma:
S → aBA | bA | a
A → ab | AbA
B → b

Se agregan nuevas producciones así:
S → CBA | BA | a
A → CB | ABA
B → b
C → a

Eliminando los triples no terminales.
S → CD | BA | a
A → CB | AD
B → b
C → a
D → BA

Ejemplo. Encontrar una GIC en FNC equivalente a la siguiente gramática:
S → AB | aBC | SBS
A → aA | C
B → bbB | b
C → cC | λ

Procedimiento.
  1. Eliminar los símbolos no terminables.
  2. Eliminar los símbolos no alcanzables.
  3. Eliminar las producciones λ (excepto S → λ si existe).
  4. Eliminar las producciones unitarias.
  5. Substituir producciones combinadas introduciendo nuevas producciones de la forma A → a.
  6. Eliminar producciones con más de dos no terminables introduciendo nuevas producciones de la forma A → BC.
Solución.
  1. Eliminar los símbolos no terminables.
        TERM1 = { S}
    TERM2 = { S,B,C,A }
    Símbolos no terminables: { }
  2. Eliminar los símbolos no alcanzables.
        Símbolos alcanzables:
    ALC1: {S}
    ALC2: {S, A}
    ALC3: {S, A, B}
    LC4: {S, A, B, C}
    Símbolos no alcanzables: { }
  3. Eliminar las producciones λ (excepto S → λ si existe).

        Variables anulables:
    ANUL1: {C}
    ANUL2: {C} U {A} = {C, A}
    S → AB | aBC | SBS | B |aB
    A → aA | C | a
    B → bbB | b
    C → cC | c

  4. Eliminar las producciones unitarias.
    S → AB | aBC | SBS | bbB | b |aB
    A → aA | cC | a | c
    B → bbB | b
    C → cC | c

  5. Substituir producciones combinadas introduciendo nuevas producciones de la forma A → a.
    S → AB | DBC | SBS | EEB | b |DB
    A → DA | FC | a | c
    B → EEB | b
    C → FC | c
    D → a
    E → b
    F → c

  6. Eliminar producciones con más de dos no terminables introduciendo nuevas producciones de la forma A → BC.
    S → AB | DI | SH | GB | b |DB
    A → DA | FC | a | c
    B → GB | b
    C → FC | c
    D → a
    E → b
    F → c
    G → EE
    H → BS
    I → BC

Aquí la gramática ya se encuentra en FNC.
