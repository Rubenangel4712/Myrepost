Conceptos de Herramientas I

  1. Escriba un programa que permita llenar un vector (arreglo unidimensional) desde el teclado indicando también su tamaño y, en una segunda parte, indicar si un número entrado también por teclado se encuentra en el vector y cuántas veces. Buscar_Uno();
  2. Escriba un programa que permita llenar un vector (arreglo unidimensional) desde el teclado indicando también su tamaño y, en una segunda parte, buscar el mayor y el menor elemento. Buscar_Dos();
  3. Escriba un programa que llene un vector de números aleatorios entre 0 y 100 y llame una función que calcula la media. Imprima el valor de la media y las posiciones en que se encuentra.

                static void  numeros_aleatorios(){
                  int numero;
                  Scanner sc = new Scanner(System.in);
                  String [] array = new String[100];// Se crea un arreglo
                                                    // con cien elementos.
                   // En esta sección se llena el arreglo.
                  for (int i = 0; i < array.length; i++) {
                      array[i] = String.format("%.0f",100*Math.random());;
                  }
                  // En esta sección se muestra el arreglo.
                  for (int i = 0; i < array.length; i++) {
                      System.out.println(" " + i + " " + array[i]);
                  }
                  // Deben aparecer cien números enteros aleatorios entre cero y 99
                  //Ejercicio: determinar si un número entrado por teclado
                  //se encuentra o no en el anterior arreglo.
                  // Si sí, indicar en qué posición se encuentra.
                  System.out.println("Entre el número a buscar: ");
                  numero = sc.nextInt();
                  for (int i = 0; i < array.length; i++) {
                      if(Integer.parseInt(array[i]) == numero){
                      System.out.println("El número se encuentra "
                              + "en la posición  " + i);
                      break;
                      }
                  }
              }
